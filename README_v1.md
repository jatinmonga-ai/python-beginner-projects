#1. Calculator

This is a beginner Python calculator project. It performs basic arithmetic operations like:
- Addition
- Subtraction
- Multiplication
- Division

## How to use
1. Download or clone the repository.
2. Run the `calculator.py` file in Python.
3. Follow the on-screen instructions to perform calculations


#2. Password Generator

This is a simple Python program that generates a strong random password based on the length you choose.
You can continue generating new passwords or type "exit" to quit.

Features
Generate one secure password at a time
Choose your own password length
Exit anytime
Beginner-friendly and clean code

## How to use
1. Download or clone the repository.
2. Run the `password_generator.py` file in Python.
3. Follow the on-screen instructions to perform calculations.
