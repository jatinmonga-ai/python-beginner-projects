#4. Rock–Paper–Scissors Game 

A simple and fun Rock–Paper–Scissors game built using Python.
This beginner-friendly project teaches basic programming concepts like user input, loops, conditionals, and the random module.

 #Features:
  - Play Rock, Paper, Scissors against the computer
  - Computer chooses moves randomly
  - Game shows who wins each round
  - Option to exit anytime
  - Clean and easy-to-understand Python code

## How to Use
1. Download or clone the repository.
2. Run the `rock_paper_scissor.py` file in python.
3. Choose one option:
   Rock : Paper: Scissors :Exit the game
4. The computer will pick a random move
5. The winner is displayed instantly
