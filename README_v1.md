#5. Emoji Chatbot

##  What This Chatbot Does

* Greets the user.
* Responds with different emojis based on mood or keywords.
* Ends when the user types bye.

##  How to Run This Project

### 1. Open IDLE (Python)

* Click **File ‚Üí New File**
