# Expense Tracker App

A simple and interactive Expense Tracker application built with Python. This app helps users manage their daily expenses, categorize them, and view total spending efficiently.

## Features
- Add Expenses: Input your daily expenses with amount, category, and description.
- View Expenses: See a list of all your recorded expenses.
- Categorize Expenses: Organize expenses under categories like Food, Transport, Entertainment, etc.
- Calculate Totals: Get total expenses at a glance.

## How to run :
1. Clone the repository
2. Run the application `expense_tracker.py`
3. Choose from the menu:
- Add an expense
- View expenses
- View total spending
- Exit
4. Follow prompts to enter details like amount, category.
