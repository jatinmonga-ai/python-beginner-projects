# To-do list app

A simple beginner-friendly Python program that allows you to manage your tasks in the console.
You can add tasks, view all tasks, delete tasks, and exit the program.

## Features:
-Add tasks to your to-do list
-View all current tasks
-Delete tasks by their number
-Exit the program at any time
-Clean and simple Python code for beginners

## How to use
1. Download or clone the repository.
2. Run the `todo_list.py` file in python.
3. Follow the on-screen menu to manage your tasks.
